STEP 2: Deploy Website
   Choose one:
   A) GitHub Pages (Free, easy)
      - Repository Settings → Pages → Select branch
      - Live at: yourusername.github.io/repo-name

   B) Cloudflare Pages (Free, fast, custom domain support)
      - Connect GitHub repo at pages.cloudflare.com
      - Add custom domain from GoDaddy
      - Automatic HTTPS and CDN

   C) Netlify (Free, automatic deployments)
      - Import from GitHub at netlify.com
      - Drag-and-drop deployment option available

STEP 3: Setup Formspree Contact Form
   → See: FORMSPREE_SETUP.md for step-by-step guide
   → Create free account at formspree.io
   → Get Form ID and update index.html (line 314)
   → Configure email recipients in Formspree dashboard
   → Test the form!

STEP 4: Connect Custom Domain (Optional)
   → Point GoDaddy nameservers to Cloudflare
   → Add domain in Cloudflare Pages
   → Wait for DNS propagation (24-48 hours)
